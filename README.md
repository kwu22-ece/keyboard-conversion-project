# Thrifted Mechanical Keyboard Mod

## Project Goal
Modify and improve a thrifted hot-swappable mechanical keyboard by upgrading its PCB, acoustics, and connectivity. The current goal of the project is changing the Mini-USB output circuit and explore conversion to USB-C, with possible future Bluetooth and signal processing support.

## Current Hardware
- Hot-swappable mechanical keyboard PCB / 3-pin switch sockets
- Old MX Blue switches
- Mini-USB output board
- Frame of keyboard

## Planned Mods
- Stabilizer tuning
- Case foam (PE and EVA Foam)
- Tape mod (Masking Tape)
- Switch/keycap replacement
- Mini-USB to USB-C output
- Possible Bluetooth and signals expansion

## Electrical Work
- Trace existing connector pins
- Identify VBUS, D+, D-, GND, shield
- Compare Mini-USB and USB-C wiring requirements
- Design or adapt USB-C daughterboard
