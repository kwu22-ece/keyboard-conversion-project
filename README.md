# Thrifted Mechanical Keyboard Mod

## Project Goal
Modify and improve a thrifted hot-swappable mechanical keyboard by upgrading its feel, sound, and connectivity. The main electrical goal is to study the existing Mini-USB output circuit and explore conversion to USB-C, with possible future Bluetooth support.

## Current Hardware
- Hot-swappable mechanical keyboard
- MX Blue switches
- 3-pin switch sockets
- Mini-USB output board
- Disassembled PCB, switches, and keycaps

## Planned Mods
- Stabilizer tuning
- Case foam
- Tape mod
- Switch/keycap replacement
- Mini-USB to USB-C investigation
- Possible Bluetooth expansion

## Electrical Work
- Trace existing connector pins
- Identify VBUS, D+, D-, GND, shield
- Compare Mini-USB and USB-C wiring requirements
- Design or adapt USB-C daughterboard
